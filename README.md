## ğŸ§© OOP Concepts Applied

### **Encapsulation**
Player attributes such as **respect**, **fear**, and **courage** are kept private in the `GameState` class and modified only through methods like `addRespect()` and `addFear()`.

### **Inheritance**
An abstract `Path` class contains shared structures used by all story paths.  
`Path1`, `Path2`, and `Path3` extend this class and implement their own unique `playPath()` logic.

### **Polymorphism**
Each path uses the same `playPath()` method name but implements different story flows, resulting in distinct narratives per player choice.

### **Abstraction**
Classes like `DelayPrinter` and `LoadingBar` hide complex animation or printing logic, giving cleaner usage throughout the program.

Follow these steps to run the program:

1. **Install Java Compiler**  
   Ensure that a Java compiler (JDK) is installed on your computer. If not, download and install the latest version of the Java Development Kit (JDK).
   
2. Create or open a folder where you want to store your Java program files.

3. Save the program with the filename:

4. Load the Java source code using your Java compiler and open the terminal.

5. Run the Program

6. **Follow On-Screen Instructions**
